# FaceID

FaceID is used to detect the general age of someone

## Installation

All you need to do is use [GIT](https://git-scm.com/) to download it!

```bash
git clone https://github.com/EvanBurnett/FaceID
```

## Usage
All you have to do is run it!

```python
python api.py
```
